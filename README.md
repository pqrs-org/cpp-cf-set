# cpp-cf-set

A utility for `CFSet`.

## Requirements

cpp-cf-set depends the following classes.

* [pqrs::cf::cf_ptr](https://github.com/pqrs-org/cpp-cf-cf_ptr).

## Install

Copy `include/pqrs` and `vendor/vendor/include` directories into your include directory.
